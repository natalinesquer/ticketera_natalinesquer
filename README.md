*** Generador de contraseñas ****
Funciona para generar contrasñas con longitud de minimo 3 caracteres hasta 50.
Escribir la longitud deseada.
Tocar boton Generar
Y trae una muy buena opcion de copiado en portapapeles.


*** Ticketera ***
Esta ticketera viene con la funcionalidad para crear un nuevo ticket, completando con datos solicitados como nombre, sector, titulo del problema y por último para que pueda detallar el mismo. Cuando el ticket sea generado se asigna un número aleatorio de solicitud de incidencia para que el usuario también pueda consultar el ticket generado.

Trae un menu donde podes elegir entre dar un alta, leer ticket y salir.
Al presionar opcion 1 tendras que completar los datos solicitados.
Se procese a asignar un número de incidencia y luego consulta si queres generar otro ticket.
En caso de no elegir generar un nuevo ticket volvera al menu inicial
Podes copiar el numero y luego elegir la opcion de leer y pegar el numerito.
Te va a mostrar los detalles de tu solicitud.
